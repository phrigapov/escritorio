# 🚀 Guia de Início Rápido

## Para Começar AGORA:

### Opção 1: Script Automático (Recomendado)

**Linux/Mac:**
```bash
./start.sh
```

**Windows:**
```
start.bat
```

### Opção 2: Manual

**Terminal 1:**
```bash
cd server
npm install
npm run dev
```

**Terminal 2:**
```bash
npm install
npm run dev
```

**Navegador:**
```
http://localhost:3000
```

## 🎮 Como Testar o Multiplayer

1. Abra o projeto em um navegador: `http://localhost:3000`
2. Digite um nome e entre
3. Abra outra aba ou outro navegador
4. Digite outro nome e entre
5. Você verá os dois avatares se movendo em tempo real!

## ⌨️ Controles

- **WASD** ou **Setas** para mover
- **Chat** no canto inferior direito para conversar

## 📝 Dicas

- Abra em múltiplas abas para simular vários usuários
- Use navegadores diferentes para testar
- Teste o chat enquanto move os avatares
- Cada jogador tem uma cor única aleatória

## ❓ Problemas?

**"Cannot find module..."**
→ Execute `npm install` na raiz e em `server/`

**"Port 3000/3001 already in use"**
→ Mate o processo ou mude a porta nos configs

**Chat não funciona**
→ Verifique se o servidor está rodando na porta 3001
